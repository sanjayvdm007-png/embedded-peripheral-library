# embedded-peripheral-library

A collection of ready-to-use Arduino/embedded C code snippets for common 
peripherals like OLED displays, servo motors, sensors and more.

## Peripherals

| Peripheral | Description |
|------------|-------------|
| [LCD](./LCD) | LCD display examples |
| [OLED](./OLED) | OLED display code & address finder |
| [SERVO](./SERVO) | Servo motor control |
